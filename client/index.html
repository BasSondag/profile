about me

Hi, I'm Bas Sondag - I moved to the San Francisco Bay Area in 2015. In my previous life I was a manager of an optical store and contactlens specialist for 16 years with the chain Eye Wish Opticiens in the Netherlands.

Immersing myself in a new country was not exciting enough. I also wanted to become a professional software engineer and make a complete career switch. Upon further research I enrolled in Coding Dojo's Onsite Flex Bootcamp.

In 1100 hours I became a fully functional developer in MEAN, Python and Ruby on Rails. It gave me a pretty amazing feeling that I can create web applications from front end to back end all by myself. Check out my bootcamp projects.

Recently I have worked on projects for Warpcache, Brigadehub and Letters4Animals. I also work as a freelance developer. Looking for developer support on a project? Send me an email.

Secrets

Secrets

Secrets is a Ruby on Rails project. You can share your biggest secrets anonymously with the members of the app, as well as read and like other members' secrets. This project started as an assignment to learn more about TDD.

All the features are built with tests written in RSpec. After it was finished I added Bootstrap and CSS. Technologies used: HTML5, CSS3, Ruby on Rails, PostgreSQL, Active Record, RSpect, Font Awesome, BCrypt, Bootstrap.

Source code: https://github.com/BasSondag/secrets

The Game

The Game

The Game was my final 2-person project for MEAN Stack built in 4 days. It is a game where one player is shown a name or object and must give hints to the other player to guess what it is. While the basis for the game is finished, we want to be able to play the game through the chat using Sockets.io.

Responsible for front end chat and the local login through Passport, back end controllers and models, Node.js server MongoDB and Package.json. Technologies used: HTML5, CSS3, JavaScript, Node.js, MongoDB, Mongoose, Passport, jQuery, Angular, Socket.io, Express, Font Awesome, Bcrypt, Bootstrap.

Source code: https://github.com/BasSondag/the_game
