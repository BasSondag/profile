about me

Hi, I'm Bas Sondag - I've recently moved from Utrecht, the Netherlands to San Jose, United States. In the Netherlands I was a manager of an optical store and contactlens specialist for 16 years with the chain Eye Wish Opticiens.

Immersing myself in a new country was not exciting enough. I also wanted to learn to code and make a complete career switch. Upon further research I enrolled in Coding Dojo's Onsite Flex Bootcamp.

In 1100 hours I became a fully functional developer in MEAN, Python and Ruby on Rails. It's a pretty amazing feeling that I can create web applications from front end to back end all by myself.

LunchBuddys

Lunch Buddys

LunchBuddys won the Coding Dojo hackathon in the Python stack. It connects professionals based on their LinkedIn profiles. A new way to expand your network, or to connect similar thinking minds over lunch.

I was responsible for front-end HTML5/CSS, jQuery, Bootstrap, MySQL Database and the MySQL queries in this 2-person project. Technologies used: HTML5, CSS3, Python, Flask, MySQL, jQuery, OAuth 2.0, Bootstrap.

Secrets

Secrets

Secrets is a Ruby on Rails project. You can share your biggest secrets anonymously with the members of the app, as well as read and like other members' secrets. This project started as an assignment to learn more about TDD.

All the features are built with tests written in RSpec. After it was finished I added Bootstrap and CSS. Technologies used: HTML5, CSS3, Ruby on Rails, PostgreSQL, Active Record, RSpect, Font Awesome, BCrypt, Bootstrap.

The Game

The Game

The Game was my final 2-person project for MEAN Stack built in 4 days. It is a game where one player is shown a name or object and must give hints to the other player to guess what it is. While the basis for the game is finished, we want to be able to play the game through the chat using Sockets.io.

Responsible for front end chat and the local login through Passport, back end controllers and models, Node.js server MongoDB and Package.json. Technologies used: HTML5, CSS3, JavaScript, Node.js, MongoDB, Mongoose, Passport, jQuery, Angular, Socket.io, Express, Font Awesome, Bcrypt, Bootstrap.
